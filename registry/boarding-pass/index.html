Boarding Pass

A flight boarding pass: passenger, carrier, flight, route, departure, seat, boarding group, and disclosable PNR.

SD-JWT VC vct
https://schemas.glideidentity.app/vct/boarding-pass
JSON-LD @context
https://schemas.glideidentity.app/contexts/boarding-pass.jsonld

Claims

Claim Type Required Selectively disclosable Notes
passenger_name string Yes always
carrier string Yes allowed IATA airline designator (2 chars).
flight_number string Yes allowed Carrier code + numeric flight number.
origin string <iata-airport> Yes allowed IATA airport code of departure.
destination string <iata-airport> Yes allowed IATA airport code of arrival.
departure_time string <date-time> Yes allowed Scheduled departure (local time with offset).
seat string No allowed
boarding_group string No allowed
pnr string No always Passenger Name Record / booking reference. Disclosable.

Auto-derived from schema.json + SD-JWT VC type metadata. always = issuer MUST make selectively disclosable; allowed = MAY; — = not individually disclosable.
