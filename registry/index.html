Schema registry

8 credential types, generated from schemas/registry.json. Each is available as W3C VC 2.0 JSON-LD and SD-JWT VC.

W3C VCDM 2.0 SD-JWT VC draft-16 RFC 9901 JSON Schema 2020-12

  • Employee ID

    v1.0.0

    Employer-issued workforce credential: employer, employee ID, role, department, and employment status.

    https://schemas.glideidentity.app/vct/employee-id

  • Proof of Employment

    v1.0.0

    Verifiable 'employed at' claim (current or former / alumni) without exposing salary or compensation.

    https://schemas.glideidentity.app/vct/proof-of-employment

  • Bank ID

    v1.0.0

    Bank-issued identity credential with verified KYC level and jurisdiction. Carries a national-ID linkage flag, never the national ID itself.

    https://schemas.glideidentity.app/vct/bank-id

  • Phone Number Verification

    v1.0.0

    Cryptographic proof of control over an E.164 phone number (not a live OTP).

    https://schemas.glideidentity.app/vct/phone-verification

  • Email Verification

    v1.0.0

    Proof of control over an email address.

    https://schemas.glideidentity.app/vct/email-verification

  • Age Verification

    v1.0.0

    Proves a holder's age via boolean age-over predicates and/or a selectively disclosable birthdate. Designed so a holder can present only a predicate (e.g. age_over_18) without revealing birthdate.

    https://schemas.glideidentity.app/vct/age-verification

  • Education Certificate

    v1.0.0

    An academic credential: institution, degree, field of study, award date, optional grade, and accreditation body.

    https://schemas.glideidentity.app/vct/education-certificate

  • Boarding Pass

    v1.0.0

    A flight boarding pass: passenger, carrier, flight, route, departure, seat, boarding group, and disclosable PNR.

    https://schemas.glideidentity.app/vct/boarding-pass
