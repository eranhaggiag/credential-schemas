Bank ID

Bank-issued identity credential with verified KYC level and jurisdiction. Carries a national-ID linkage flag, never the national ID itself.

SD-JWT VC vct
https://schemas.glideidentity.app/vct/bank-id
JSON-LD @context
https://schemas.glideidentity.app/contexts/bank-id.jsonld

Claims

Claim Type Required Selectively disclosable Notes
issuing_institution string Yes allowed Name of the issuing financial institution.
account_holder_name string Yes always Verified legal name of the account holder.
kyc_level enum(basic | standard | enhanced) Yes allowed Verified Know-Your-Customer assurance level.
jurisdiction string Yes allowed ISO 3166-1 alpha-2 country code of the regulating jurisdiction.
national_id_linked boolean No allowed Whether a national ID was verified and linked at the bank. The national ID itself is NEVER included.

Auto-derived from schema.json + SD-JWT VC type metadata. always = issuer MUST make selectively disclosable; allowed = MAY; — = not individually disclosable.
